## 🎯 What is Dockronos? / ¿Qué es Dockronos?

**English:** Dockronos is a powerful TUI application designed to simplify container management in development environments. It provides an intuitive, keyboard-driven interface for managing Docker and Podman containers with real-time monitoring, log streaming, and configuration management.

**Español:** Dockronos es una potente aplicación TUI diseñada para simplificar la gestión de contenedores en entornos de desarrollo. Ofrece una interfaz intuitiva y orientada al teclado para gestionar contenedores Docker y Podman con monitoreo en tiempo real, transmisión de logs y gestión de configuración.

---

## ✨ Key Features / Características principales

### English
- **Interactive TUI** — Three-panel layout with keyboard navigation  
- **Multi-engine Support** — Works with Docker and Podman  
- **Real-time Monitoring** — Live metrics and log streaming  
- **Configuration Management** — YAML-based project setup  
- **Cross-platform** — Binaries available for major platforms  
- **Keyboard Shortcuts** — Efficient Vi-style navigation  
- **Template System** — Pre-configured container templates  
- **Git Integration** — Repository management and auto-discovery  

### Español
- **TUI interactiva** — Diseño de tres paneles con navegación por teclado  
- **Soporte multi-motor** — Funciona con Docker y Podman  
- **Monitoreo en tiempo real** — Métricas en vivo y transmisión de logs  
- **Gestión de configuración** — Proyectos basados en YAML  
- **Multiplataforma** — Binarios para las principales plataformas  
- **Atajos de teclado** — Navegación eficiente estilo Vi  
- **Sistema de plantillas** — Plantillas de contenedores preconfiguradas  
- **Integración con Git** — Gestión de repositorios y autodescubrimiento
